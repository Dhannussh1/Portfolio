# ♟️ Dhannussh R - Personal Portfolio

A premium, interactive, chess-themed personal portfolio showcasing my journey, technical skills, projects, and achievements.

## 🚀 Features

- **Chess-Themed Design**: Custom King-piece cursor and a sophisticated color palette inspired by the game of kings.
- **Interactive Journey**: A high-impact "Life Journey" timeline with smooth sliding transitions.
- **Dynamic Projects**: Grid of major projects with tech stacks and glassmorphic hover effects.
- **Academic Scoreboard**: A custom-designed, high-contrast scoreboard for academic and co-curricular achievements.
- **Family Tree**: A heartfelt tribute to my roots and the silent pillars of my life.
- **Mobile Responsive**: Fully optimized for mobile devices with touch swipe support for sliders and a responsive hamburger menu.
- **Custom Lightbox**: Click any image to view it full-screen with smooth blur transitions.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS with advanced grid/flex layouts and custom animations.
- **JavaScript**: Native ES6+ for custom sliders, cursor logic, and intersection observers.
- **FontAwesome**: For premium icons.

## 📱 Mobile Support
The site includes specialized touch-event handling for mobile browsers, ensuring that all sliders and interactive elements respond naturally to swipes.
